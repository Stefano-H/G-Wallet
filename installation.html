INSTALLATION

VirtualBox Installation and OVA Import Manual

Installing VirtualBox:

Downloading VirtualBox:

  • Go to the VirtualBox website: https://www.virtualbox.org/
  • Click on "Downloads" in the navigation menu.
  • Choose the version for your operating system

Installation

  • Run the downloaded installation file.
  • Follow the installation wizard instructions.
  • Once the installation is complete, VirtualBox will be ready to use.


Importing the GWallet system's OVA file into VirtualBox:

Downloading the GWallet OVA file:

  • Make sure you have the OVA file of the GWallet system that you want to import. It can be a file downloaded from the internet or provided by a developer.

Opening VirtualBox:

  • Open VirtualBox from the start menu or application launcher, depending on your operating system.

Importing the OVA file:

  • In the VirtualBox main window, click on "File" in the top-left corner and select "Import Appliance...".
  • The import wizard will open. Click on "Choose a file" and navigate to the GWallet system's OVA file you want to import.
  • Once the file is selected, click "Next" and then "Import".
  • VirtualBox will start importing the OVA file. This process may take a few minutes, depending on the file size and your system's speed.

Completion:

  • Once the import is complete, you will see the virtual appliance in the list of virtual machines in VirtualBox.
  • Select the newly imported virtual machine and click "Start" to begin using it.


That's it! You have successfully installed VirtualBox and imported the GWallet system's OVA file to use it in a virtual machine. Now you can start working with your virtual environment as needed.
